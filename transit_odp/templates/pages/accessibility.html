{% trans 'Accessibility statement' %}

{% blocktrans %} This statement applies to pages on bus-data.dft.gov.uk. It does not apply to the whole of gov.uk. {% endblocktrans %}

{% blocktrans %} This website is run by the Department for Transport. The text should be clear and simple to understand. You should be able to: {% endblocktrans %}

  • {% blocktrans %} zoom in up to 300% without problems {% endblocktrans %}
  • {% blocktrans %} navigate all of the website using just a keyboard (except for maps which are not essential to the functionality) {% endblocktrans %}
  • {% blocktrans %} navigate all of the website using speech recognition software {% endblocktrans %}
  • {% blocktrans %} use all of the website using a screen reader (including the most recent versions of JAWS, NVDA and VoiceOver) {% endblocktrans %}
  • {% blocktrans %} interpret page information through access technology due to a consistent heading structure {% endblocktrans %}
  • {% blocktrans %} use all of the functionality with Javascript turned off {% endblocktrans %}
  • {% blocktrans %} access the site and use the associated services with Google Chrome, Internet Explorer, Safari, Opera, Firefox, and Edge {% endblocktrans %}

{% trans 'How to request content in an accessible format' %}

{% blocktrans with link="BusOpenData@dft.gov.uk" %} If you need information in a different format then please contact us at {{link}} and tell us: {% endblocktrans %}

  • {% blocktrans %} the web address (URL) of the content {% endblocktrans %}
  • {% blocktrans %} your name and email address {% endblocktrans %}
  • {% blocktrans %} the format you need, for example, audio CD, braille, BSL or large print, accessible PDF {% endblocktrans %}

{% trans 'Reporting accessibility problems with this website' %}

{% blocktrans with link="BusOpenData@dft.gov.uk"%} If you find any problems that are not listed on this page or you think we're not meeting the accessibility requirements, then please contact us at {{link}}. {% endblocktrans %}

{% trans 'Enforcement procedure' %}

{% blocktrans with link="https://www.equalityadvisoryservice.com/"%} If you contact us with a complaint and you're not happy with our response contact the Equality Advisory and Support Service (EASS). {% endblocktrans %}

{% blocktrans %} The Equality and Human Rights Commission (EHRC) is responsible for enforcing the Public Sector Bodies (Websites and Mobile Applications) (No. 2) Accessibility Regulations 2018 (the 'Accessibility Regulations'). {% endblocktrans %}

{% trans "Technical information about this website's accessibility" %}

{% blocktrans %} The Department for Transport is committed to making its websites accessible, in accordance with the Accessibility Regulations. {% endblocktrans %}

{% blocktrans with link="https://www.w3.org/TR/WCAG21/" %} This website follows the Web Content Accessibility Guidelines version (WCAG) 2.1 AA standard. It is successfully XHTML 1.0 strict valid. {% endblocktrans %}

{% trans 'Non-accessible content' %}

{% blocktrans %} The content that is not accessible is outlined below: {% endblocktrans %}

  • {% blocktrans %} Our maps cannot be directly read by a screen reader (but key information is in the text and enlarged print versions can be provided) {% endblocktrans %}
  • {% blocktrans %} Keyboard only users will not always be able to zoom on our maps {% endblocktrans %}

{% trans 'How we tested this website' %}

{% blocktrans with link="http://digitalaccessibilitycentre.org/index.php/kpmg-publish-bus-open-data"%} This website is tested against the Web Content Accessibility Guidelines V2.1 level A and level AA. These tests have been carried out both internally and with the Digital Accessibility Centre. {% endblocktrans %}
