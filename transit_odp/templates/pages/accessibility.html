{% trans 'Accessibility statement' %}

{% filter force_escape %} {% blocktrans %} This statement applies to pages on bus-data.dft.gov.uk. {% endblocktrans %} {% endfilter %}

{% filter force_escape %} {% blocktrans %} This website is run by the Department for Transport. The text should be clear and simple to understand. You should be able to: {% endblocktrans %} {% endfilter %}

  • {% filter force_escape %} {% blocktrans %} zoom in up to 300% without problems {% endblocktrans %} {% endfilter %}
  • {% filter force_escape %} {% blocktrans %} navigate all of the website using just a keyboard (except for maps which are not essential to the functionality) {% endblocktrans %} {% endfilter %}
  • {% filter force_escape %} {% blocktrans %} navigate all of the website using speech recognition software {% endblocktrans %} {% endfilter %}
  • {% filter force_escape %} {% blocktrans %} use all of the website using a screen reader (including the most recent versions of JAWS, NVDA and VoiceOver) {% endblocktrans %} {% endfilter %}
  • {% filter force_escape %} {% blocktrans %} interpret page information through access technology due to a consistent heading structure {% endblocktrans %} {% endfilter %}
  • {% filter force_escape %} {% blocktrans %} use all of the functionality with Javascript turned off {% endblocktrans %} {% endfilter %}
  • {% filter force_escape %} {% blocktrans %} access the site and use the associated services with Google Chrome, Internet Explorer, Safari, Opera, Firefox, and Edge {% endblocktrans %} {% endfilter %}

{% filter force_escape %}{% trans 'How accessible this website is' %}{% endfilter %}

  • {% filter force_escape %} {% blocktrans %} We know parts of this website is inaccessible. They include: The map function which isn’t readily accessible by the screen reader and the key board function won’t pick it up directly (please read below for more details) {% endblocktrans %} {% endfilter %}
